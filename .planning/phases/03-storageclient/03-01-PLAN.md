   Struct fields:
   - `auth_fetch: Mutex<AuthFetch<W>>` — tokio::sync::Mutex (NOT std), per STATE.md locked decision
   - `endpoint_url: String`
   - `next_id: AtomicU64` — initialized to 1
   - `settings: Mutex<Option<Settings>>` — tokio::sync::Mutex for async access
   - `settings_cached: AtomicBool` — for sync `is_available()`, initialized false

   Generic bound: `W: WalletInterface + Clone + Send + Sync + 'static`

   Constructor `pub fn new(wallet: W, endpoint_url: impl Into<String>) -> Self` — creates AuthFetch::new(wallet), wraps in Mutex, sets all fields.

   Private `async fn rpc_call<T: serde::de::DeserializeOwned>(&self, method: &str, params: Vec<Value>) -> WalletResult<T>`:
   - `let id = self.next_id.fetch_add(1, Ordering::Relaxed);`
   - Build JSON-RPC 2.0 envelope: `{ "jsonrpc": "2.0", "method": method, "params": params, "id": id }`
   - Serialize to `Vec<u8>` via `serde_json::to_vec`
   - Create headers HashMap with `"content-type": "application/json"`
   - Lock auth_fetch, call `fetch(&self.endpoint_url, "POST", Some(body_bytes), Some(headers)).await`
   - Map AuthFetch error to `WalletError::Internal(format!("auth fetch: {}", e))`
   - Check `response.status >= 400` -> return HTTP error
   - Parse response body as `serde_json::Value`
   - Check `json.get("error")` -> deserialize as WalletErrorObject -> `wallet_error_from_object`
   - Extract `json["result"]` -> `serde_json::from_value::<T>` -> return
