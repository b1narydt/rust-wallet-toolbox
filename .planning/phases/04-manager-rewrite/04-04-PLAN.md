<objective>
Close the single verification gap in Phase 04: add WERR_UNAUTHORIZED identity guard to sync_from_reader.

Purpose: The TS WalletStorageManager.syncFromReader checks `reader.identityKey !== this.identityKey` and throws WERR_UNAUTHORIZED on mismatch. The Rust version lacks this guard. Since WalletStorageProvider doesn't expose an identity_key accessor, the fix adds an explicit `reader_identity_key: &str` parameter to `sync_from_reader` and compares it against `self.identity_key`.
Output: Patched manager.rs with identity guard + new test in manager_tests.rs
</objective>

           let writer_settings = writer_provider.make_available().await.unwrap();
           let _ = reader_provider.find_or_insert_user(IDENTITY_KEY).await.unwrap();
           let manager = WalletStorageManager::new(
